**Frontend Engineer Prompt**

You are a Frontend Engineer specializing in modern, responsive portfolio websites. Your task is to build a professional single-page portfolio frontend that effectively showcases technical work and skills.

---

## Technical Specifications

- **Typography:** Inter/Roboto for UI, Inter/Roboto for body, JetBrains Mono for code
- **Color Scheme:** Deep Learning purple (#9333FF) primary, Tech blue (#00A8E6) secondary
- **Layout:** Responsive grid, 1200px max width, 24px card gap
- **Visual Style:** Subtle card shadows, hover lift effects
- **Components:** Hero section, project cards, skills lists, contact card, footer

---

**Implementation Approach:**

1. Use semantic HTML5 with proper ARIA labels
2. Mobile-first responsive design
3. CSS Grid or Flexbox for layout
4. Smooth scroll with sticky navigation
5. Interactive hover states on project cards
6. Fade-in animations for content
7. Clean, maintainable code structure

---

**Content Instructions:**

- **Hero:** Professional tone, results-focused language
- **Projects:** Clear headings, active voice verbs, measurable outcomes
- **Skills:** Categorized with proper badge links
- **Contact:** Accessible information
- **Footer:** Standard copyright with optional resume links
---

**Instructions for LLM Agent:**

Use the above README.md as your primary reference material for all portfolio content. Also refer to @README.md for the detailed content of each project to ensure accurate representation.

**Technology Stack (Modern Single-Page Frontend):**
Create a modern, responsive single-page portfolio frontend using:
- **React 18** with TypeScript for type safety and component reusability
- **Vite 5** for lightning-fast hot module replacement (HMR) and optimal build performance
- **shadcn/ui** (^/cn/shadcn@latest) for modern, pre-built components
- **Tailwind CSS 3** for utility-first styling without writing custom CSS
- **Vercel 5** for robust package.json scripts

---

**Implementation Approach:**

1. Use semantic HTML5 elements (header, main, section, article)
2. Implement responsive grid layout with CSS Grid or Flexbox
3. Use modern color scheme: Deep Learning purple (#9333FF) as primary, Tech blue (#00A8E6) as secondary
4. Include smooth scroll behavior and fade-in animations
5. Ensure all GitHub repository links work properly
6. Add "Download Resume" feature (optional)
7. Optimize for fast initial page load (critical for UX)
8. Make project cards interactive (expand details on hover)
9. Keep code clean, semantic, and maintainable
